# HyperMind 官网

基于 Next.js 14 + TypeScript + Tailwind CSS 构建的现代化官网。

## 开发

安装依赖：
```bash
npm install
```

运行开发服务器：
```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看网站。

## 构建

```bash
npm run build
npm start
```

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## 功能特性

- 响应式设计
- 现代化 UI/UX
- 优化的性能
- SEO 友好
